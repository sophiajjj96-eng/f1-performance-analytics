# Tableau Dashboard

This folder contains the Tableau workbook used to visualize the results from the SQL analysis and KPI modeling.

The dashboard translates the analytical metrics into visual insights that allow easier comparison of driver and constructor performance.


## Dashboard Overview

The Tableau dashboard explores multiple aspects of Formula 1 performance including driver success, race outcomes, and constructor trends.

The visualizations focus on performance comparisons across drivers and teams.


## Key Visualizations

**Driver Performance Overview**

* Total career points by driver
* Race wins comparison
* Average starting grid position

**Constructor Performance**

* Podium finish distribution by constructor
* Team ranking comparison
* Constructor performance variability


## Metrics Visualized

The dashboard is based on KPI metrics generated through SQL analysis, including:

* total career points
* race wins
* average starting position
* constructor variability
